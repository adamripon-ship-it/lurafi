# Multi-language (12 locales)

Registry: `config/languages.json`  
Helpers: `scripts/i18n/registry.mjs`

## Locales

| Code | Prefix | Shopify locale |
|------|--------|----------------|
| en | `/` | en (primary) |
| nl | `/nl` | nl |

## Theme strings

```bash
node scripts/build-locales.mjs          # en.default.json + nl.json from section sources
DEEPL_API_KEY=xxx node scripts/translate-locales.mjs   # fr.json, de.json, …
node scripts/translate-locales.mjs --skip-existing     # only missing files
```

Glossary terms (`Kevin`, `Kevin+`, `Lurafi`, …) are preserved during machine translation.

shopify store auth --store fu03cn-1v.myshopify.com \

## Adding a 13th language

1. Add an entry to `config/languages.json` (handles, `ogLocale`, `urlPrefix`).
2. `node scripts/build-locales.mjs && node scripts/translate-locales.mjs --locale=xx`
3. `./scripts/activate-locales.sh`
4. Regenerate sitemap/LLM assets and push theme `locales/` folder.
